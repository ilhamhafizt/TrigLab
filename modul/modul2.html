Modul 2 —
Grafik Fungsi Trigonometri

Modul ini membahas bagaimana grafik fungsi sinus, cosinus, dan tangen berubah ketika amplitudo, periode, dan translasi dimodifikasi.

Pelajari Konsep Mulai Eksplorasi

Konsep Dasar Fungsi Trigonometri

1. Hubungan dengan Unit Circle

Fungsi sinus dan cosinus berasal dari lingkaran satuan (unit circle), yaitu lingkaran berjari-jari 1 dengan pusat di (0, 0).

  • cos(x) = Koordinat x pada lingkaran
  • sin(x) = Koordinat y pada lingkaran
  • tan(x) = sin(x) / cos(x)

Ketika sudut bertambah, titik bergerak mengelilingi lingkaran, dan proyeksinya membentuk grafik gelombang.

2. Bentuk Umum Fungsi

y = A sin(Bx + C)

y = A cos(Bx + C)

y = A tan(Bx + C)

3. Makna Setiap Parameter

  • B (Periode / Frekuensi)
    Mengatur panjang satu gelombang penuh.
    Periode fungsi sinus dan cosinus:
    T = 2π / B
    Semakin besar B → grafik semakin rapat.
  • C (Translasi / Phase Shift)
    Menggeser grafik ke kiri atau kanan.
    Pergeseran = −C/B

4. Periode dan Frekuensi

Periode adalah panjang satu siklus lengkap. Untuk fungsi datar:

  • sin(x) dan cos(x) → periode = 2π
  • tan(x) → periode = π

Jika B berubah, maka periode menjadi: T = 2π/B (untuk sin dan cos)

5. Perbandingan Grafik

Sinus

  • Mulai dari (0, 0)
  • Naik ke maksimum
  • Gelombang halus

Cosinus

  • Mulai dari maksimum
  • Bergeser 90° dari sinus
  • Gelombang halus

Tangen

  • Tidak memiliki amplitudo tetap
  • Memiliki asimtot vertikal
  • periode lebih pendek (π)

6. Interpretasi Visual dalam Eksplorasi

Saat Anda mengubah slider:

  • Amplitudi → tinggi gelombang berubah.
  • Periode → kerapatan gelombang berubah.
  • Translasi → grafik bergeser horizontal.

Perhatikan bagaimana perubahan kecil pada parameter dapat menghasilkan perubahan signifikan pada grafik.
